Unfollow your Unfollowers!

Sarthak Srivastava

3 min read
·
Mar 4

Instagram has been around for quite some time now. A lot of features and UI keeps changing over time. In one of the recent changes, Instagram has removed the feature to collectively list the people who dont follow you back! The only way to check the same is by manually going on every profile and checking their followings.

Well, the good news is all this can be easily done which just using few lines of codes and in a manner where your data is completely secured (without the involvement of any third-party app) as all the processing happens on your local machine. Lets get to it, for better understanding of the steps involved, I have added the flowchart:

Step 1: Download your personal data from Instagram.

Once the data is downloaded, you can unzip the data into a folder, the two most important files needed for this can be found inside a folder called ‘followers_and_following’.

Step2: Set up your working environment. You can run the code using Conda locally or on Google Colab.

  1. Open a Jupyter notebook or Google Colab and install the prerequisites.

2. Upload the files from the previous step in your current working directory. You can use the command os.getcwd() method to get current working directory.

3. Scrape the webpage using beautifulsoup and create a list of followers and following:

#Opening files from the current working directory. 

file1 = open('followers_1.html', 'r')
file2 = open('following.html', 'r')

#Defining a function for scraping the HTML document and returing the lists of followers and following. 

def list_creator(file1=file1, file2=file2):
    
    list1 = [] #for followers
    list2 = [] #for following
    
    doc=BeautifulSoup(file1, 'html.parser')
    tag = doc.find_all('a')
    for tags in tag:
        list1.append(tags.text)   
     
    print('Total number of followers: {}'.format(len(list1)))


    doc=BeautifulSoup(file2, 'html.parser')
    tag = doc.find_all('a')
    
    for tags in tag:
        list2.append(tags.text)
    
    print('Total number of following: {}'.format(len(list2)))   
    
    return(list1, list2)

4. Use a comparator function to compare the lists created by the ‘list_creator’ function.

def comparator(fr,fg):
    
    dont_follow_back = []
    c=0
    
    print("Followers: {} ".format(len(fr)))
    print("Following: {} ".format(len(fg)))
    
    for f in fg:
        if f not in fr:
            dont_follow_back.append(f)
            c = c+1

            
    print('There are {} accounts which you follow, but they dont follow you back :('.format(c))
    
    return(dont_follow_back)
